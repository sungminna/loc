# Template design concepts & strategy

9 reel templates, each with a distinct visual language and a clear
retention strategy. Eight are slide-based (Remotion-rendered card
news with rich animation); the ninth is video-based, generated by
Seedance 2.0 and overlaid with infographic typography.

The principle: **시선을 끌되 과하지 않게**. Every template avoids
infinite-jiggle and disco-strobe noise — motion serves the message.

## Video template (Seedance 2.0)

| # | Slug | Composition | Inspiration | When to pick it | Retention hook |
|---|---|---|---|---|---|
| 10 | `seedance-reel` | `SeedanceReel` | TikTok / Reels 시네마틱 | 스토리텔링, 라이프 다큐, 제품 무비 | 영상 자체의 motion + 인포그래픽 오버레이 = 정보 + 감정 |

각 씬은 Replicate `bytedance/seedance-2.0`로 생성 → R2에 업로드 →
SeedanceReel이 클립을 이어붙이며 위에 챕터 라벨, 키네틱 헤드라인,
글래스 stat 카드, 진행 바, 워터마크를 얹는다. 스토리라인 아크는
`HOOK → PAYOFF → ACTION → CLOSE` (3-5씬, 총 18-25s).

지원하는 Seedance 2.0 입력:
- 텍스트 → 비디오 (가장 가성비)
- 첫 프레임 (gpt-image-2) → 비디오 (캐릭터/색감 일관성)
- 첫 + 마지막 프레임 (제어된 컷)
- 레퍼런스 이미지 ≤9장 (캐릭터 일관성, 다중 샷)
- 레퍼런스 영상 ≤3개 (모션 트랜스퍼)
- 레퍼런스 오디오 ≤3개 (립싱크)
- `generate_audio` 자동 BGM/대사 (BGM 트랙과 ducking 처리)

## Animation primitives

`src/remotion/animations.ts`가 공유 빌딩 블록을 제공:

- `springStagger` — 자식들을 N프레임씩 어긋나게 등장시키는 스프링
- `splitTextProgress` — 단어/글자 단위 reveal 진행도
- `maskWipe` — 좌/우/상/하 wipe (clip-path)
- `kenBurns` — 슬로우 줌+팬
- `countUp` — 숫자 0→target 카운트업
- `marquee` — 무한 가로 흐름
- `slideEnvelope` — 슬라이드 enter/exit 한 줄로
- `chromaShadow`, `grainPulse` — 레트로 효과

새 템플릿을 추가할 때 이 모듈을 import해서 1-3줄로 풍부한 모션을
만들 수 있다.

## Retention design rules

각 템플릿이 공유하는 원칙:

1. **첫 1초에 시선 잡기** — 슬라이드 1은 더 큰 폰트, 더 강한 모션, 강조 색.
2. **진행도 시각화** — 진행 바 또는 인덱스(01/05) 항상 노출 → "끝까지 보면 5초만 더".
3. **시각적 대비** — 큰 숫자(stat) ↔ 작은 라벨, 두꺼운 헤드라인 ↔ 가는 본문.
4. **CTA는 마지막 슬라이드 한 곳에만** — `SAVE / TRY / SHARE` 중 하나.
5. **모션은 0.5-1초 안에 멈춘다** — 본문이 정적 상태로 읽히는 시간이 충분해야 한다. 계속 움직이는 텍스트는 읽기 어려워 retention을 깎는다.
6. **음악 ducking** — 영상 템플릿은 클립 오디오 0.85 + BGM 0.18로 자동 ducking.
7. **타이포그래피가 정보** — 배경 이미지는 0.3-0.5 opacity 폴백. 텍스트가 항상 명료하게 읽힌다.

## How to add another template

1. `src/remotion/compositions/<MyTpl>.tsx` 작성 (`CardSlideProps` or `VideoReelProps` 구현)
2. `src/remotion/Root.tsx`에 `<Composition id="MyTpl" .../>` 등록 + `SLIDE_FRAMES_BY_COMP`에 항목 추가
3. `scripts/seed.ts`의 `templateInsert(...)` 호출 추가 (또는 dashboard에서 바로 추가)
4. 토픽에서 그 슬러그 선택

Sandbox는 매 실행마다 git clone하므로 커밋 즉시 적용.
